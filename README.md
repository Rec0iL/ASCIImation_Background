# ASCIImation KDE Plasma Wallpaper & Lock Screen Plugin

A fun and nostalgic wallpaper and lock screen plugin for KDE Plasma 6 that brings classic ASCII animations straight to your desktop.

## Features

This plugin includes several iconic and newly generated ASCII animations:
* **Star Wars: Episode IV** (The classic ASCII animation by Simon Jansen)
* **Nyan Cat Rainbow Flight**
* **Party Parrot Dance**
* **3D Spinning Wireframe Cube**
* **Matrix Digital Rain**
* **ASCII Aquarium**
* **Rotating Donut (Torus)**
* **Fireworks Display**
* **DVD Bouncing Logo**
* **Sine Wave Oscillator**
* **Rotating Wireframe Globe**

You can also customize the appearance by adjusting phosphor colors, CRT scanlines, and animation speed!

## Prerequisites
* KDE Plasma 6
* Python 3
* `bash`

## Installation

1. Clone or download this repository.
2. Run the `install.sh` script to fetch the necessary datasets, generate the animation libraries, and install the plugin to your local KDE Plasma environment:

```bash
./install.sh
```

## Usage

To set it as your Lock Screen Background in KDE Plasma 6:
1. Open **System Settings** -> **Screen Locking**
2. Click on **Appearance** (Configure Lock Screen Appearance)
3. Change **Wallpaper Type** to **ASCIImation**
4. Choose your desired scene from the settings.
5. Adjust the phosphor colors, CRT scanlines, and speed to your liking, then click **Apply**!
